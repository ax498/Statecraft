## 二、核心脚本详解

### 2.1 main_menu.gd
- **路径**：`scenes/main/main_menu.gd`
- **类**：`Control` 子类
- **职责**：主菜单与多级导航（主界面 → 模式选择 → 对决）；设置中心、Mod 管理入口
- **主要方法**：
  | 方法 | 逻辑 |
  |------|------|
  | `_ready` | 连接信号、检测从对决返回，若 `return_to_mode_select` 则显示模式选择 |
  | `_on_start_game_pressed` | 切换到模式选择视图 |
  | `_on_lunce_pressed` | 弹出论策人数对话框 (2~8) |
  | `_show_lunce_player_count_dialog` | 创建 SpinBox 选择人数，确认后设置 `BattleManager.player_count`、`is_lunce_mode`，重置 CardManager/BattleManager，切到 `battle_test.tscn` |
  | `_on_yanbing_pressed` | 弹出演兵网络对话框 |
  | `_show_yanbing_network_dialog` | 创建/加入房间、IP/端口、连接信号、Mod 校验 |
  | `_populate_setting_buttons` | 按当前视图填充设置按钮（Mod 扩展、返回、退出） |
  | `_populate_mod_panel` | 填充 Mod 列表、详情、覆盖资源，连接启用/详情回调 |
- **依赖**：ModManager, BattleManager, CardManager, NetworkManager

---

### 2.2 battle_test.gd
- **路径**：`scenes/battle/battle_test.gd`
- **类**：`Control` 子类
- **职责**：战斗 UI 层；抽牌/收牌/矛盾等动画；卡牌展示；九牛一毛/出奇制胜/偷梁换柱/他山之石等选择 UI；联机客户端渲染
- **主要方法**：
  | 方法 | 逻辑 |
  |------|------|
  | `_ready` | 初始化背景、Manager、信号、布局、手牌显示、AI 回合确保 |
  | `_on_draw_pressed` / `_on_end_turn_pressed` | 转发到 `BattleManager.request_draw` / `request_end_turn` |
  | `_on_card_drawn` | 主机/单机：添加到回合区并播放抽牌动画；联机客户端跳过（由 `client_draw_received` 处理） |
  | `_play_card_draw_animation` | 滑入+缩放动画，结束后调用 `_play_card_specific_effect`（卡牌专属特效） |
  | `_animate_cards_to_hand` | 收牌飞入手牌动画，结束后 `collection_animation_finished` |
  | `_on_contradiction` | 设置保护/未保护样式、矛盾提示、震动闪红 |
  | `_play_contradiction_fade_out` | 矛盾淡出：保护牌飞入手牌，未保护牌淡出，最后 `contradiction_effect_complete` |
  | `_play_card_specific_effect` | 察言观色/九牛一毛/出奇制胜/自圆其说/见好就收/偷梁换柱/箭在弦上等专属特效 |
  | `_on_jiuniuyimao_selection_requested` | 高亮对手卡牌供选择，添加放弃按钮 |
  | `_on_jiuniuyimao_steal_animation_started` | 从对手手牌飞牌到回合区，动画结束后 `complete_jiuniuyimao_steal_animation` |
  | `_on_mod_soft_remove_effect_requested` | 他山之石删牌：发光→震动→淡出，结束后 `mod_resume_without_settle` |
  | `get_target_container` / `get_visual_target` | 按逻辑玩家索引返回手牌容器（本地/对手） |
  | `_refresh_hand_displays` | 刷新本地和对手手牌 UI |
- **依赖**：BattleManager, CardManager, ModManager, NetworkManager, GameLogger

---

### 2.3 battle_manager.gd
- **路径**：`scripts/core/battle_manager.gd`
- **类**：`Node` 单例
- **职责**：战斗状态机（START_BATTLE / PLAYER_TURN / PROCESSING / CONTRADICTION）；多玩家轮替；分数；抽牌/结束回合请求；联机主机权威、RPC 同步；AI 回合；九牛一毛/出奇制胜/偷梁换柱/他山之石选择与效果
- **状态枚举**：`State.START_BATTLE`, `State.PLAYER_TURN`, `State.PROCESSING`, `State.CONTRADICTION`
- **主要方法**：
  | 方法 | 逻辑 |
  |------|------|
  | `request_draw` | 检查 effect_playing、当前状态、回合归属；联机时主机执行、客户端 RPC；调用 CardManager.draw_card |
  | `request_end_turn` | 空回合直接 next_turn；否则 settle_turn；联机主机 RPC 结算与收牌 |
  | `draw_animation_finished` | 抽牌动画结束；处理 Mod 劫持（九牛一毛/出奇制胜/偷梁换柱/他山之石）；若无劫持则切换到 PLAYER_TURN，AI 则启动 AI 回合 |
  | `next_turn` | 计算新索引，联机主机 RPC `_rpc_sync_turn` |
  | `collection_animation_finished` | 收牌动画完成，执行回合切换 |
  | `contradiction_effect_complete` | 矛盾特效完成，执行 next_turn |
  | `request_jiuniuyimao_selection` | 九牛一毛选择，夺取牌、飞牌动画、联机同步 |
  | `request_chuqizhisheng_selection` | 出奇制胜选择，弃牌特效 |
  | `request_toulianghuanzhu_selection` | 偷梁换柱选择，交换动画，触发被选牌效果 |
  | `request_trigger_card_effect` | 按 ability_id 触发察言观色/九牛一毛/出奇制胜/见好就收/箭在弦上/偷梁换柱/Mod 钩子 |
  | `mod_resume_without_settle` | 恢复流程不结算，继续抽牌 |
  | `complete_stop_card_effect` | 见好就收特效结束，强制结算 |
  | `complete_jiuniuyimao_steal_animation` | 九牛一毛飞牌结束，判断矛盾后触发被夺取牌效果 |
- **依赖**：CardManager, ModLoader, NetworkManager, GameLogger

### 2.4 card_manager.gd
- **路径**：`scripts/core/card_manager.gd`
- **类**：`Node` 单例
- **职责**：牌堆构建；抽牌；矛盾检测；回合结算；手牌/回合区/弃牌堆管理
- **主要方法**：
  | 方法 | 逻辑 |
  |------|------|
  | `_build_deck` | 从 CardRegistry 与 mod_registered_cards 构建 global_deck，洗牌 |
  | `draw_card` | pre_draw_card 钩子→从牌堆顶抽牌→post_draw_card 钩子（他山之石可消费）→矛盾检测→触发卡牌效果→emit card_drawn |
  | `settle_turn` | 结算前检测矛盾；累加 effect_value 到 player_scores；移入 player_hands；turn_settled 信号 |
  | `trigger_contradiction` | 自圆其说保护逻辑；未保护牌进弃牌堆；emit contradiction_triggered |
  | `check_name_contradiction` | 检测 turn_container 同名卡牌 |
  | `transfer_card_from_hand_to_turn` | 九牛一毛：从手牌移至回合区 |
  | `remove_card_from_hand` | 他山之石/出奇制胜：从手牌移除并入弃牌堆 |
  | `peek_next_card` | 察言观色：查看牌堆顶 |
  | `apply_draw_from_data` | 联机客户端：按主机数据生成显示 |
  | `apply_settlement_result` | 联机客户端：按主机结算结果应用 |
- **依赖**：CardRegistry, BattleManager, ModLoader, GameLogger

### 2.5 card_ui.gd

  | `set_contradiction_style` | 矛盾时红色 |
  | `set_protected_style` | 自圆其说保护时蓝色 |
  | `play_transform_animation` | 神来之笔变身动画 |
  | `play_extra_action_glow` | 额外行动发光循环 |
  | `set_compact` | 手牌区缩小显示 |
- **依赖**：CardResource

### 2.6 card_registry.gd
- **路径**：`scripts/cards/card_registry.gd`
- **类**：`RefCounted`
- **职责**：核心卡牌注册表；提供 `get_all_core_cards()` 供 CardManager 构建牌堆
- **CARD_SCRIPTS**：card_null, card_stop, card_double, card_chayanguanse, card_jiuniuyimao, card_chuqizhisheng, card_ziyuanqishuo, card_jianzaixianshang, card_toulianghuanzhu
- **主要方法**：
  | 方法 | 逻辑 |
  |------|------|
  | `get_all_core_cards` | 遍历 CARD_SCRIPTS，加载并调用 `get_registration_data()`，返回注册数据数组 |
  | `get_log_format_for_card` | 根据卡牌匹配注册表，返回 get_log_format 结果 |
- **依赖**：各 card_*.gd

### 2.7 card_*.gd（核心卡牌脚本）

均继承 `RefCounted`，实现 `get_registration_data()` 与可选 `get_log_format()`。

| 文件 | 卡牌名 | ability_id | 简要效果 |
|------|--------|------------|----------|
| card_null.gd | 空牌 | "" | 无效果，不参与矛盾 |
| card_stop.gd | 见好就收 | stop | force_ends_turn=true，抽到即结算 |
| card_double.gd | 一石二鸟 | double | 双倍分值 |
| card_chayanguanse.gd | 察言观色 | chayanguanse | 查看下一张牌 |
| card_jiuniuyimao.gd | 九牛一毛 | jiuniuyimao | 夺取对手牌 |
| card_chuqizhisheng.gd | 出奇制胜 | chuqizhisheng | 弃对手最高分牌 |
| card_ziyuanqishuo.gd | 自圆其说 | ziyuanqishuo | 矛盾时保护前序牌 |
| card_jianzaixianshang.gd | 箭在弦上 | jianzaixianshang | 强制再抽一张 |
| card_toulianghuanzhu.gd | 偷梁换柱 | toulianghuanzhu | 交换回合区卡牌并触发被选牌效果 |

### 2.8 autoload 与 core 目录

#### game_logger.gd
- **路径**：`scripts/autoload/game_logger.gd`
- **职责**：缓冲日志、每局保存到 logs/、安卓暂停时刷新
- **主要方法**：`record`, `flush`, `save_session`, `_prune_old_logs`

#### mod_loader.gd
- **路径**：`scripts/core/mod_loader.gd`
- **职责**：扫描 Mod 目录 .gd 文件、实例化、`_mod_init`、`_on_hook_triggered`、`trigger_hook`
- **主要方法**：`trigger_hook`, `api_call`, `replace_node_script`, `reload_mod_scripts`

#### mod_manager.gd
- **路径**：`scripts/core/mod_manager.gd`
- **职责**：Mod 目录管理、mod_info.json、启用状态持久化、资源覆盖检测、联机 Mod 校验
- **主要方法**：`get_mod_storage_path`, `reload_all_mods`, `get_mod_list_for_verification`, `verify_mod_list_contains`, `generate_example_mod_template`

#### network_manager.gd
- **路径**：`scripts/core/network_manager.gd`
- **职责**：局域网 ENet 创建/加入房间、Mod 校验、场景同步、重开
- **主要方法**：`create_game`, `join_game`, `start_yanbing_game`, `disconnect_peer`, `request_exit_game`, `request_restart_game`, `is_multiplayer`, `get_local_player_index`

### 2.10 Mod：他山之石 (tashanzhishi_mod/main.gd)

- **职责**：抽到他山之石时，若任意一方手牌有一石二鸟，分两次选择（一石二鸟 + 对手牌），被选牌移入弃牌堆
- **钩子**：post_draw_card, sync_tashanzhishi_mode, post_draw_animation, stolen_card_effect, trigger_card_effect, tashanzhishi_selection_finished
- **流程**：`_mod_init` 注册卡牌 → post_draw_card 劫持 → post_draw_animation 高亮选择 → 选择完成后 `request_mod_soft_remove_effect`

## 三、模块调用关系概览

```
main_menu
  ├─ ModManager (mod_details, set_mod_enabled, reload_all_mods)
  ├─ BattleManager (is_lunce_mode, player_count, return_to_mode_select)
  ├─ CardManager (reset_for_new_game)
  └─ NetworkManager (create_game, join_game, start_yanbing_game)

battle_test
  ├─ BattleManager (request_draw, request_end_turn, 各种信号)
  ├─ CardManager (player_hands, turn_container, remove_card_from_hand)
  ├─ ModManager (mod_details, generate_example_mod_template)
  └─ NetworkManager (is_multiplayer, get_local_player_index)

BattleManager
  ├─ CardManager (draw_card, settle_turn, turn_container, player_hands)
  ├─ ModLoader (trigger_hook: post_draw_animation, pre_turn_settled, post_turn_settled)
  └─ NetworkManager (is_multiplayer, opponent_id, RPC)

CardManager
  ├─ CardRegistry (get_all_core_cards)
  ├─ BattleManager (request_trigger_card_effect, check_score_contradiction)
  └─ ModLoader (trigger_hook: pre_draw_card, post_draw_card)
```

## 四、关键信号流向

| 信号源 | 信号 | 接收方 | 用途 |
|--------|------|--------|------|
| CardManager | card_drawn | battle_test | 回合区显示 + 抽牌动画 |
| CardManager | contradiction_triggered | BattleManager | 矛盾流程 |
| BattleManager | hand_collection_requested | battle_test | 收牌动画同步 |
| BattleManager | jiuniuyimao_selection_requested | battle_test | 九牛一毛选择 UI |
| BattleManager | mod_soft_remove_effect_requested | battle_test | 他山之石删牌特效 |
| BattleManager | draw_animation_finished | BattleManager 内部 | 抽牌动画结束，推进流程 |
